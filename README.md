# CS50 - Zelda

Repositório com a tarefa do Legend of Zelda do Curso CS50's Introduction to Game Development onde os objetivos são:

- Implementar corações que caem dos inimigos aleatóriamente quando abatidos e que regeneram um coração completo do jogador quando coletados
- Adicionar potes aleatóriamente que o jogador pode agarrar (Com as sprites respectivas). Quando agarrando o pote o jogador não pode atacar com a espada e deve arremessar o pote quebrando caso acerte um inimigo causando dano, viaje 4 casas ou atinja uma parede
-

Repositório original do curso: https://github.com/games50/zelda

#  Controles
- Iniciar: **ENTER**
- Movimentação: **Setas**
- Atacar, agarrar e arremessar potes: **ESPAÇO**
